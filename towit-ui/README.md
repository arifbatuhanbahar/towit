# Towit UI Prototype Structure

- `mobile/`: mobile prototype entry files (`Towit.html`, `app.jsx`, iOS/design canvas files)
- `web/`: web prototype entry files (`Towit Web.html`, `web-app.jsx`)
- `components/`: shared UI components used by both prototypes
- `styles/`: shared style files used by both prototypes

Open:
- Mobile: `towit-ui/mobile/Towit.html`
- Web: `towit-ui/web/Towit Web.html`
